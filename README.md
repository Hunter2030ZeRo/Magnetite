# Magnetite

Magnetite is an open-source web browser based on Servo, built upon Tauri and SolidJS frontend and Rust backend. 

## Why is it different from the others?

Most modern web browsers such as Chrome(Chromium), Microsoft Edge and Firefox are often heavy and resource-intensive, showing suboptimal resource consumption. 
This causes significant RAM consumption and battery drainage, making it especially unsuitable for usage in laptops. To mitigate this, Magnetite aims to minimize resource usage, 
creating an ideal portable browsing experience for everyone.
